## ğŸ“¥ Installation

1. Download the latest release from [Releases](../../releases) or [Planet Minecraft](https://www.planetminecraft.com/data-pack/spectral-frequency/)
2. Place the `spectral_frequency` folder in your world's `datapacks` folder
